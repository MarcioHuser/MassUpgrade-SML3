A mod to mass upgrade a series of connected belts and lifts.

This mod is still experimental. If you are having crashes after commanding upgrades/downgrades, disable it and contact me at Satisfactory Modding Discord server



This mod is completely free. But if you wish to donate a value as a “thank you”, feel free to click on the button below. Find me at Satisfactory Modding Discord server (@MarcioHuser)

How to use it:

  • A Hand Tool to allow you to aim and select a belt or lift the belongs to the system you want to upgrade. It can be unlocked in the HUB at Tier 5
  • Uppon selecting a belt/lift, a poup will be presented with all the available belt and lift speeds you have already unlocked
  • It will list all kind of belts and lifts in the system, with refundable items and necessary cost to make the upgrad
  • If you want to track the necessary resources, you can click at the "Add to List" button to to the To-Do list
  • You can choose if you want to traverse through Splitters, Mergers, Storage Teleporters, Modular Load Ballancers, Underground Belts, Train Cargo Platoforms, Pipe Junctions and Power Switches
  • If you have all the necessary resources for the selected recipes, the "Upgrade" button will be enabled
